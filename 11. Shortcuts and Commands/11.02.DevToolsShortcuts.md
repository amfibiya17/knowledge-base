# 11.02 DevTools Shortcuts (Windows only)

## Open / Close

| Action                     | Shortcut           |
| -------------------------- | ------------------ |
| Open DevTools              | `F12`              |
| Open DevTools (dock right) | `Ctrl + Shift + I` |
| Toggle DevTools            | `Ctrl + Shift + I` |
| Close DevTools             | `Ctrl + Shift + I` |

---

## Panels

| Panel       | Shortcut           |
| ----------- | ------------------ |
| Elements    | `Ctrl + Shift + C` |
| Console     | `Ctrl + Shift + J` |
| Sources     | `Ctrl + Shift + S` |
| Network     | `Ctrl + Shift + E` |
| Performance | `Ctrl + Shift + P` |
| Application | via panel menu     |

---

## Elements panel

| Action               | Shortcut           |
| -------------------- | ------------------ |
| Inspect element      | `Ctrl + Shift + C` |
| Edit as HTML         | `F2`               |
| Toggle element state | `:hov` button      |
| Force hover / focus  | `:hov`             |

---

## Console

| Action            | Shortcut        |
| ----------------- | --------------- |
| Clear console     | `Ctrl + L`      |
| Multi-line input  | `Shift + Enter` |
| Previous command  | `↑`             |
| Next command      | `↓`             |
| Run selected code | `Ctrl + Enter`  |

---

## Sources (Debugging)

| Action                | Shortcut          |
| --------------------- | ----------------- |
| Pause / resume script | `F8`              |
| Step over             | `F10`             |
| Step into             | `F11`             |
| Step out              | `Shift + F11`     |
| Toggle breakpoint     | Click line number |

## Useful combos

| Goal                   | Action              |
| ---------------------- | ------------------- |
| Debug JS + inspect DOM | Elements + Sources  |
| Check API calls        | Network → Fetch/XHR |
| Test mobile layout     | Device toolbar      |
| Check errors fast      | Console             |
