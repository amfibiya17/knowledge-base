# 11.01 VSCode Shortcuts (Windows only)

## General

| Action              | Shortcut            |
| ------------------- | ------------------- |
| Command Palette     | `Ctrl + Shift + P`  |
| Quick Open (file)   | `Ctrl + P`          |
| Settings            | `Ctrl + ,`          |
| Toggle Terminal     | `Ctrl + \``         |
| New Terminal        | `Ctrl + Shift + \`` |
| Toggle Sidebar      | `Ctrl + B`          |
| Toggle Zen Mode     | `Ctrl + K Z`        |
| Split Editor        | `Ctrl + \`          |
| Switch Editor Group | `Ctrl + 1 / 2 / 3`  |

---

## Editing

| Action               | Shortcut                  |
| -------------------- | ------------------------- |
| Multi-cursor (add)   | `Alt + Click`             |
| Select next match    | `Ctrl + D`                |
| Select all matches   | `Ctrl + Shift + L`        |
| Expand selection     | `Shift + Alt + Right`     |
| Shrink selection     | `Shift + Alt + Left`      |
| Move line up/down    | `Alt + Up / Down`         |
| Copy line up/down    | `Shift + Alt + Up / Down` |
| Delete line          | `Ctrl + Shift + K`        |
| Toggle line comment  | `Ctrl + /`                |
| Toggle block comment | `Shift + Alt + A`         |
| Format document      | `Shift + Alt + F`         |
| Format selection     | `Ctrl + K Ctrl + F`       |
| Rename symbol        | `F2`                      |
| Quick Fix            | `Ctrl + .`                |

---

## Search & Navigation

| Action           | Shortcut           |
| ---------------- | ------------------ |
| Find             | `Ctrl + F`         |
| Replace          | `Ctrl + H`         |
| Find in files    | `Ctrl + Shift + F` |
| Go to line       | `Ctrl + G`         |
| Go to symbol     | `Ctrl + Shift + O` |
| Go to definition | `F12`              |
| Peek definition  | `Alt + F12`        |
| Go back          | `Alt + Left`       |
| Go forward       | `Alt + Right`      |

---

## File / Editor

| Action               | Shortcut           |
| -------------------- | ------------------ |
| New file             | `Ctrl + N`         |
| Save                 | `Ctrl + S`         |
| Save all             | `Ctrl + K S`       |
| Close editor         | `Ctrl + W`         |
| Reopen closed editor | `Ctrl + Shift + T` |
