# 5.09 CSS Flexbox Layout Guide

Flexbox is a CSS layout system for arranging items in **one direction**:
a row **or** a column.

Use it for:

- nav bars
- button rows
- centering
- aligning items
- simple responsive layouts

---

## Mental model

- **flex container** = the parent (`display: flex`)
- **flex items** = the children
- Flexbox is best for **1D layout** (row OR column)
- Grid is best for **2D layout** (rows AND columns)

---

## Create a flex container

```css
.container {
  display: flex;
}
```

Default behaviour:

- direction: row
- items stay on one line (no wrap)

---

## Direction (row vs column)

```css
.container {
  display: flex;
  flex-direction: row; /* default */
}
```

```css
.container {
  display: flex;
  flex-direction: column;
}
```

---

## Main axis vs cross axis (!)

Axis depends on `flex-direction`:

### If `flex-direction: row`

- **main axis** = left → right
- **cross axis** = top → bottom

### If `flex-direction: column`

- **main axis** = top → bottom
- **cross axis** = left → right

---

## Aligning items

### `justify-content` (main axis)

Controls spacing **along the main axis**.

```css
.container {
  display: flex;
  justify-content: flex-start; /* default */
}
```

Common values:

```css
justify-content: flex-start;
justify-content: center;
justify-content: flex-end;
justify-content: space-between;
justify-content: space-around;
justify-content: space-evenly;
```

---

### `align-items` (cross axis)

Controls alignment **across the cross axis**.

```css
.container {
  display: flex;
  align-items: stretch; /* default */
}
```

Common values:

```css
align-items: stretch;
align-items: center;
align-items: flex-start;
align-items: flex-end;
```

---

## Gap

Use `gap` for spacing between items (cleaner than margins).

```css
.container {
  display: flex;
  gap: 12px;
}
```

---

## Wrapping

Without wrap, items try to stay on one line.

```css
.container {
  display: flex;
  flex-wrap: wrap;
}
```

Useful for:

- tags
- card lists
- responsive grids (simple ones)

---

## Common patterns

### 1) Center something (horizontal + vertical)

```css
.center {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

---

### 2) Space between (left and right items)

```css
.toolbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
}
```

---

### 3) Navigation row

```css
.nav {
  display: flex;
  gap: 16px;
  align-items: center;
}
```

---

### 4) Simple sidebar layout

```css
.page {
  display: flex;
}

.sidebar {
  width: 240px;
}

.content {
  flex: 1;
}
```

---

## Flex sizing (children)

### `flex: 1` (fill remaining space)

```css
.item {
  flex: 1;
}
```

Equivalent idea:

- grow = 1
- shrink = 1
- basis = 0

---

### Fixed + flexible items

```css
.left {
  width: 200px;
}

.right {
  flex: 1;
}
```

---

### Prevent shrinking

```css
.logo {
  flex-shrink: 0;
}
```

Useful for:

- logos
- icons
- fixed-size buttons

---

## Align one item differently

### `align-self` (single item)

```css
.item {
  align-self: center;
}
```

---

## Common mistakes

- mixing up `justify-content` vs `align-items`
- forgetting axis changes when switching row/column
- using margins instead of `gap`
- using Flexbox for 2D grid layouts (use Grid)

---

## Quick cheat sheet

| Goal                             | Use               |
| -------------------------------- | ----------------- |
| horizontal spacing in a row      | `justify-content` |
| vertical alignment in a row      | `align-items`     |
| vertical spacing in a column     | `justify-content` |
| horizontal alignment in a column | `align-items`     |
| spacing between items            | `gap`             |
| wrap onto next line              | `flex-wrap: wrap` |
| make item fill space             | `flex: 1`         |

---

## Summary

- Flexbox = 1D layout (row OR column)
- Parent controls layout with `display: flex`
- `justify-content` = main axis
- `align-items` = cross axis
- Use `gap` and `wrap` for clean spacing and responsiveness
