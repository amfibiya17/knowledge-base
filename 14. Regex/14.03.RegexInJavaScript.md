# 14.03 Regex Groups, Alternation & Flags

Common regex **building blocks** used to combine patterns.

---

## Groups

| Use case            | Regex     |
| ------------------- | --------- |
| Capturing group     | `(abc)`   |
| Non-capturing group | `(?:abc)` |
| Optional group      | `(abc)?`  |
| Repeated group      | `(abc)+`  |

---

## Alternation (OR)

| Use case            | Regex           |
| ------------------- | --------------- |
| This OR that        | `cat|dog`       |
| Grouped alternation | `(jpg|png)`     |
| Multiple options    | `(one|two|3)`   |

---

## Optional parts

| Use case           | Regex     |
| ------------------ | --------- |
| Optional character | `colou?r` |
| Optional word      | `(test)?` |
| Optional group end | `https?`  |

---

## Lookarounds

| Use case                 | Regex     |
| ------------------------ | --------- |
| Followed by (lookahead)  | `a(?=b)`  |
| Not followed by          | `a(?!b)`  |
| Preceded by (lookbehind) | `(?<=a)b` |
| Not preceded by          | `(?<!a)b` |

---

## Anchors (position-based)

| Use case        | Regex |
| --------------- | ----- |
| Start of string | `^`   |
| End of string   | `$`   |
| Word boundary   | `\b`  |
| Not boundary    | `\B`  |

---

## Flags / modifiers

| Flag | Meaning              |
| ---- | -------------------- |
| `i`  | Case-insensitive     |
| `g`  | Global (all matches) |
| `m`  | Multiline            |
| `s`  | Dot matches newline  |
| `u`  | Unicode              |

Example:

```js
// match "hello" globally (all occurrences), case-insensitive
/hello/gi;
```

---

## Common combinations

| Use case              | Regex              |
| --------------------- | ------------------ |
| Case-insensitive word | `^hello$` + `i`    |
| File types            | `\.(jpg|png|gif)$` |
| Optional protocol     | `https?:\/\/`      |
| Multiple separators   | `[,\s]+`           |

---

## Quantifiers

| Symbol  | Meaning     |
| ------- | ----------- |
| `?`     | 0 or 1      |
| `*`     | 0 or more   |
| `+`     | 1 or more   |
| `{n}`   | Exactly n   |
| `{a,b}` | Between a–b |

---

## Mental shortcut

- `()` → group things
- `|` → OR
- `? * +` → how many
- Flags → how regex behaves
