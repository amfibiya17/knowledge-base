# 14.02 Common Regex Patterns

Reusable regex patterns for common cases.

---

## Digits & numbers

| Use case         | Regex        |
| ---------------- | ------------ |
| Only digits      | `^\d+$`      |
| Exactly 3 digits | `^\d{3}$`    |
| 7–15 digits      | `^\d{7,15}$` |
| Integer (±)      | `^-?\d+$`    |

---

## Letters & words

| Use case               | Regex           |
| ---------------------- | --------------- |
| Only letters           | `^[A-Za-z]+$`   |
| Lowercase only         | `^[a-z]+$`      |
| Letters + spaces       | `^[A-Za-z\s]+$` |
| Word characters (`\w`) | `^\w+$`         |

---

## Email (basic)

| Use case | Regex                        |
| -------- | ---------------------------- |
| Email    | `^[^\s@]+@[^\s@]+\.[^\s@]+$` |

---

## Password rules (used separately)

| Rule                 | Regex     |
| -------------------- | --------- |
| Min 8 chars          | `^.{8,}$` |
| At least 1 digit     | `\d`      |
| At least 1 uppercase | `[A-Z]`   |
| At least 1 lowercase | `[a-z]`   |

---

## URLs

| Use case     | Regex            |
| ------------ | ---------------- |
| HTTP / HTTPS | `^https?:\/\/.+` |

---

## Whitespace

| Use case          | Regex   |
| ----------------- | ------- |
| Leading spaces    | `^\s+`  |
| Trailing spaces   | `\s+$`  |
| Multiple spaces   | `\s+`   |
| No spaces allowed | `^\S+$` |

---

## File extensions

| Use case         | Regex             |
| ---------------- | ----------------- |
| `.jpg` or `.png` | `\.(jpg|png)$`    |
| `.pdf` only      | `\.pdf$`          |

---

## Search helpers

| Use case | Regex                      |
| -------- | -------------------------- |
| Numbers  | `\d+`                      |
| Words    | `\w+`                      |
| Emails   | `[^\s@]+@[^\s@]+\.[^\s@]+` |

---

## Anchors & helpers

| Symbol  | Meaning         |
| ------- | --------------- |
| `^`     | Start of string |
| `$`     | End of string   |
| `.`     | Any character   |
| `*`     | 0 or more       |
| `+`     | 1 or more       |
| `{n}`   | Exactly n times |
| `{a,b}` | Between a and b |
